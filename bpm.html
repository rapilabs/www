Rigrr BPMN Editor

Rigrr is a free BPMN editor available either online at http://rigrr.rapilabs.com or by installing from the Chrome Web Store. It saves its process models to files using the BPMN 2.0 XML serialisation format.

Rigrr is a serverless web application that utilises HTML5 technologies to achieve a higher degree of user experience than traditional web applications. Because it does not rely on a server to operate, Rigrr is 100% offline capable.

Library Distribution

A JavaScript library that is capable of being embedded within a 3rd party application is available on a per-request basis.  To find out more please contact rigrr@rapilabs.com.
